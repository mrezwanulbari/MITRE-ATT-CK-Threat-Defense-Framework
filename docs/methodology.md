# Methodology

## 1. Technique Prioritization
Techniques are prioritized based on:
- Threat intelligence relevance
- Industry attack trends
- Financial sector risk profiles
- Cloud adoption patterns

## 2. Defense Mapping
Each technique is mapped to:
- **Preventive controls**
- **Detective controls**
- **Response actions**
- **Recovery steps**

## 3. Implementation Targets
Mappings are designed for:
- Microsoft Sentinel
- Splunk Enterprise
- XSOAR / Sentinel SOAR
- EDR/XDR platforms
- Cloud-native security tools

## 4. Validation
Detections and playbooks undergo:
- Unit testing
- Log simulation
- Adversary emulation
